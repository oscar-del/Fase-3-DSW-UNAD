OVI HTML 5


--INTRODUCCIÓN---

HTML es un lenguaje del cual se compone de elementos, son aplicados a piezas de texto para darles un significado, estructura e incrusta contenido como imágenes,videos, audios, PDF, etc en una o varias páginas.

Objetivo general:

Fortalecer los conocimientos en HTML5 con el fin de generar bases para el desarrollo de páginas web.

Objetivos específicos:

  • Estructura básica de html5 (etiquetas, atributos y valores).

  • Definir la estructura básica página HTML (HTML, HEAD, BODY).

  • Comprender el contenido en HTML ( SECTION, ASIDE, FOOTER, ARTICLE).

  • Conocer y aprender sobre CSS su importancia y aplicación
